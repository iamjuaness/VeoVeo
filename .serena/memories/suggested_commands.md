# CineMate Suggested Commands

- **Workspace Dev/Start**:
  - `pnpm dev` - Start dev servers for both frontend and backend concurrently.
- **Run Backend Tests**:
  - `pnpm --filter @veoveo/server test` - Run backend tests once.
  - `pnpm --filter @veoveo/server test -- --coverage` - Run backend tests with coverage.
- **Run Frontend Tests**:
  - `pnpm --filter @veoveo/web test` - Run frontend tests.
- **Build**:
  - `pnpm build` - Build all packages.
